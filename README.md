# eTiWi

eTiWi is an electronic tin whistle with a built-in VS1053B General MIDI synth. It generates real-time MIDI from breath + fingering and supports MIDI play-along by mixing live performance with backing-track MIDI inside the same synth engine.

It can also act as a USB-MIDI device for a computer or tablet. Audio outputs include headphones (primary), line-out for active speakers, and an on-board speaker output via a small amplifier.
