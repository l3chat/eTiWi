# MIDI Sources for Play-Along (Web Links)

## Goal

Find MIDI files suitable for play-along accompaniment (not only melody),
ideally with bass/chords/drums.

## Large MIDI Archives

- [BitMidi](https://bitmidi.com/): huge archive, easy browsing and direct
  downloads; quality varies.
- [Classical Archives (MIDI section)](https://www.classicalarchives.com/midi.html):
  large classical catalog; access varies.
- [Piano-midi.de](https://piano-midi.de/): generally higher-quality classical
  piano MIDIs.
- [MIDIworld (Classical)](https://www.midiworld.com/classic.htm): older but
  useful composer-organized catalog.

## Scores to MIDI Export (Often Best for Accompaniment)

- [MuseScore export guide](https://handbook.musescore.org/file-management/file-export):
  open scores (MSCZ/MusicXML) and export to MIDI.
- [MusicXML source list](https://www.musicxml.com/music-in-musicxml/): some
  sources provide MusicXML files that can be exported to MIDI via MuseScore.

## IMSLP Note

IMSLP is excellent for sheet music, but it generally does not host standard
MIDI uploads for mainstream repertoire. Use it primarily for PDFs and scores,
not as the main MIDI source.

- [IMSLP performers portal](https://imslp.org/wiki/IMSLP:Performers_Portal)

## Suggested Search Keywords

- "backing", "accompaniment", "karaoke", "minus one"
- Add style/genre terms such as "irish", "folk", "jig", "reel", and "waltz".
