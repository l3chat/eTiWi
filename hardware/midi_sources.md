# MIDI Sources for Play-Along (web links)

Goal: find MIDI files suitable for **play-along accompaniment** (not only melody), ideally with bass/chords/drums.

## Large MIDI archives
- BitMidi — huge archive, easy browsing and direct downloads; quality varies.
  https://bitmidi.com/
- Classical Archives (MIDI section) — large classical catalog; access varies.
  https://www.classicalarchives.com/midi.html
- Piano-midi.de — generally higher-quality classical piano MIDIs.
  https://piano-midi.de/
- MIDIworld (Classical) — older but useful composer-organized catalog.
  https://www.midiworld.com/classic.htm

## Scores -> export to MIDI (often best for accompaniment)
- MuseScore export: open scores (MSCZ/MusicXML) and export to MIDI.
  https://handbook.musescore.org/file-management/file-export
- MusicXML sources list (some provide MusicXML usable for MIDI export via MuseScore).
  https://www.musicxml.com/music-in-musicxml/

## IMSLP note
IMSLP is excellent for sheet music, but it generally does not host standard MIDI uploads for mainstream repertoire;
use it primarily for PDFs / scores, not as your main MIDI source.
- IMSLP performers portal:
  https://imslp.org/wiki/IMSLP:Performers_Portal
