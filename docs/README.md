# Documentation

This directory holds project-level documentation that is not limited to one
hardware subfolder.

## What Goes Here

- System-level architecture and behavior notes.
- Cross-domain plans that connect hardware and firmware work.
- Roadmaps, milestones, and decision logs.

## Current Documents

- [roadmap.md](roadmap.md): staged V1 and V2 priorities.
